# SalaryVision AI

A professional Salary Prediction system with both Web (Flask) and Desktop (Tkinter) interfaces.

## Features
- Interactive Dashboards
- - Multi-dimensional Salary Analysis
  - - Professional UI/UX
    - - Real-time Predictions
     
      - ## Execution Guide
     
      - ### 1. Prerequisites
      - Ensure you have Python installed. Install dependencies:
      - ```bash
        pip install -r requirements.txt
        ```

        ### 2. Run Web Application
        ```bash
        python app.py
        ```
        Access via http://127.0.0.1:5000

        ### 3. Run Desktop Application
        ```bash
        python salary_prediction_app.py
        ```
